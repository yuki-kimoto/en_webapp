Introduction to Perl Web App Development

Getting Started with Perl Web App Development describes how to develop web applications in Perl. Perl has a web framework called Mojolicious that makes it easy and flexible to create web applications. With Mojolicious, you can build a web system that has both development efficiency and performance. Currently, this site is almost in the process of being written.
